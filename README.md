# Snack-recommender
Project Introduction:This project is an interactive web application based on the K-Nearest Neighbors (KNN) algorithm. It connects to a real food nutrition database and uses data mining techniques to provide users with personalized healthy snack recommendations.This project is not only a practical tool, but also an EdTech case study, designed to show beginners the underlying logic of data cleaning, feature engineering, and machine learning algorithms.


 Core Functionality: KNN Intelligent Matching: Based on Euclidean distance, it finds the food closest to the user's needs from a multi-dimensional feature space (calories, protein). Data Mining Workflow Demonstration: Data Cleaning: Automatically handles missing and outlier values. Feature Normalization: Uses MinMaxScaler to eliminate the influence of different physical units on distance calculations. Outlier Injection Tutorial: Includes a classic "labeling error" case to demonstrate the importance of data quality. Algorithm Perspective: Real-time display of distance values ​​and matching scores from the algorithm's backend. 

 
Data Mining Workflow: Data Integration: Integrates the nutrient library and allergen safety library. Feature Selection: Selects calories, protein, and allergen identifiers as core features. Data Transformation: Normalizes numerical features. Pattern Discovery: Uses KNN to search for $k$ nearest neighbors.
